# Provenance Guard

Backend system for classifying creative content attribution (human vs AI-generated) with transparency labels, appeals, and audit logging.

> Full documentation will be completed in Milestone 6 after implementation.

## Setup

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env   # add your GROQ_API_KEY
python app.py
```
